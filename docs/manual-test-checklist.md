# FoodMoment 手动测试检查清单

> 在 iOS 模拟器上逐项测试，勾选已通过的项目

## 一、启动与登录

- [ ] App 能正常启动，无崩溃
- [ ] 启动页/闪屏显示正常

- [ ] 三层卡路里环形图显示
- [ ] 环形图中心数字显示剩余卡路里
- [ ] 底部三个宏量营养素指标显示
- [ ] 饮水卡片显示
- [ ] 步数卡片显示
- [ ] "今日食刻" 标题显示
- [ ] 食刻轮播卡片区域显示

### 交互检查
- [ ] 下拉可刷新
- [ ] 点击饮水卡片 "+" 按钮弹出饮水追踪
- [ ] 食刻卡片可左右滑动
- [ ] 点击 "View All" 跳转日记页
